# K-ZONE
- klab(ポートフォリオ)
- kazugramming(ブログ)
- k-step（配信システム）
上記のアプリを新しくまとめてリニューアル。

## klab(ポートフォリオ)
ポートフォリオをまとめたアプリです。
個人で作成したアプリやブログをまとめています。

## kazugramming(ブログ)
プログラミング特化型のブログです。

## k-step(配信システム)
配信システムです。
LINEBOTで好きな配信を選択して、配信を受けることができます。

## Requirement
### klab・kazugramming
- FW: Laravel 5.8/Next.js
- DB: MySQL 5.7
- DEV: Docker/Docker-compose/VSCode
- CI: CircleCI
- CD: AWS

### k-step(LINE BOT)
- FW: Ruby on Rails 5.2 || Flask
- DB: MySQL 5.7
- CI: CircleCI
- CD: AWS

## その他
- ドメイン: AWS Route53
- ホスティング: AWS S3
- サーバー: AWS EC2
- データベース: AWS RDS
- キャッシュ: AWS ElastiCache
- メール: AWS SES
- ログ: AWS CloudWatch
